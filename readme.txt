EUROC datasets are available **[here](http://projects.asl.ethz.ch/datasets/doku.php?id=kmavvisualinertialdatasets)**

KITTI datasets are available (http://www.cvlibs.net/datasets/kitti/raw_data.php)

Folder 'libDird' contains the sourece code for DIRD descriptor used for visual verification in the paper.
The paper describing the approach is
How to Learn an Illumination Robust Image Feature for Place Recognition  
Henning Lategahn and Johannes Beck and Bernd Kitt and Christoph Stiller 
IEEE Intelligent Vehicles Symposium (IV) 2013

Folder 'seqslam' contains the source code for SeqSLAM algoritm published by Milford and Wyeth at ICRA12 [1].
[1] Michael Milford and Gordon F. Wyeth (2012). SeqSLAM: Visual Route-Based Navigation for Sunny Summer Days and Stormy Winter Nights. In Proc. of IEEE Intl. Conf. on Robotics and Automation (ICRA)

Folder VIO conatains the source code used for the paper "Unscented Kalman Filtering on Lie Groups for Fusion of IMU and Monocular Vision", submitted to International Conference on Robotics and Automation (ICRA) 2018. **[PDF](https://hal.archives-ouvertes.fr/hal-01588669/document)**

Folder VIO-DIRD contains some results of the proposed algorithms in the paper.
